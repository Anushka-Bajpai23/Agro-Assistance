## 🎯 Project Overview

**Agro-Assistance** is an AI-powered Smart Farming Assistant that offers comprehensive agricultural guidance through crop recommendation, disease diagnosis, fertilizer guidance, and water usage insights. Built with a clean web interface, this platform empowers farmers with data-driven decisions to optimize their farming practices.

<table>
<tr>
<td width="50%">

### 🌾 **Problem Statement**
Traditional farming faces challenges like disease outbreaks, unpredictable yields, water wastage, and poor resource management. Farmers need intelligent tools to make data-driven decisions and optimize their agricultural practices.

</td>
<td width="50%">

### 🤖 **AI-Powered Solution**
A comprehensive Smart Farming Assistant using machine learning algorithms to provide crop recommendations, accurate disease diagnosis, fertilizer guidance, and water usage optimization through an intuitive web interface.

</td>
</tr>
</table>

## 🚀 Installation & Setup

### Prerequisites
```bash
Python 3.8 or higher
PyTorch 1.9+
pip package manager
```

### Quick Start
```bash
# Clone the repository
git clone https://github.com/Anushka-Bajpai23/Agro-Assistance.git
cd Agro-Assistance

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the Streamlit application
streamlit run app.py
```

### Required Dependencies
